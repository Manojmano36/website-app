The DNA of your Code

A Git SHA-1 hash is the unique identifier that ensures your project's history remains immutable and secure.

f7a2b3c...8e9d01f

Digital Fingerprint

A 40-character hexadecimal string generated from your commit's exact content.

Data Integrity

If even a single pixel or character changes, the entire SHA changes instantly.

Parent Tracking

Each SHA contains a reference to its parent, creating a secure chain of history.

What makes a SHA?

01

File Contents

The actual changes in your code (blobs).

02

Metadata

Author name, email, and timestamp.

03

Commit Message

The description you wrote for the change.

04

Parent Hash

The ID of the commit that came before.
